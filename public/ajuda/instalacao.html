Instalação

Para a implementação básica do Infodoc no linux ,será necessária a instalação dos seguintes aplicativos:

  • Instalação Ruby e RubyGems
  • Instalação o Rails
  • Instalação o Git
  • Instalação o Infodoc

Instalação Ruby e RubyGems

Se ainda não houver instalado o Ruby, digite na linha de comando do console:

sudo aptitude install ruby rubygems

Instalando o Rails

Digite o seguinte na linha de comando :

sudo gem install rails

Após concluído o download e a configuração da Gem , digite o seguinte:

rails -v

Se tudo estiver certo, será informada a versão do Rails instalada.

Instalando o git

O git é necessário para fazer o download direto do repositório do Infodoc.

No console do linux digite o seguinte:

apt-get install git-core

Instalando o Infodoc

Para instalar o infodoc utilize o seguinte comando:

git install git@github.com:isaacguerra/infodoc_open.git

Após o download, teste se o servidor está funcionando:

cd infodoc_open
script/server
